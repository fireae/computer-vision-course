 ## 运行测试
1. 构建并运行
```
make line # 直线检测
make circle # 圆形检测
```
或直接运行可执行文件：
```
cd src && ./cv_hw3_1
cd src && ./cv_hw3_2
```
2. 目录说明


1. docs      ：相关测试文档，以及实验报告
2. output  ：运行，测试过程中的图像输出
3. images ：输入图像
4. src         ：项目代码
5. build     ：运行时产生的中间链接文件
